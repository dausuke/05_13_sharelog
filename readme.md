# sharelog(シェアログ)
デプロイ先：https://dausuke.github.io/03_13_mylog/

## プロダクトの紹介
- 自分の行ったお店のログを記録できるサイト。
- 保存した日を記録し、いつ行ったか、お店の種類や評価などを記録。
- 保存した時点での現在地を取得。
- firestoreにデータを保存してSNSのような形で自分のログをシェアできる。

## 工夫した点
- 機能面はできるだけシンプルにして使いやすさ意識しました。
